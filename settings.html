Configured apps

How to update

Edit config.json in the project root to add or adjust apps.

Each entry should include:

  • name — tile title
  • url — destination URL
  • description — tile details
  • icon — radio or status (built-in icon keys), anything else falls back to a link icon
  • status — optional label
{
  "apps": [
    {
      "name": "Quu HD Radio Network",
      "url": "https://hdradio.quucast.com",
      "description": "Open the primary HD Radio network control panel.",
      "icon": "radio"
    }
  ]
}
